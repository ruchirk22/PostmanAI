How to Get Your Postman API Key

1. Open Postman and click your avatar in the top-right corner.

2. Go to Settings > API Keys.

3. If you have an existing key, you can use that. Otherwise, click "Generate API Key".

4. Give your key a nickname (e.g., "AI_Assistant") and click "Generate".

5. Copy the generated key and paste it into the input field. Make sure to save it somewhere safe, as you won't be able to see it again.
